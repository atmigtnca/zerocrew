## 왜 이걸 만들려고 하는지
매일 점심시간에 어떤 식당이 뭘 파는지 몰라서 돌아다니다가 시간을 허비함.
학교 홈페이지에 메뉴가 올라오긴 하는데 각 식당별로 따로 확인해야 해서 불편함.

## 현재 사람들이 이 문제를 어떻게 해결하는지
- 학교 홈페이지에서 식당별로 하나씩 확인
- 에브리타임 게시판에서 물어봄
- 그냥 매일 같은 식당만 감

## 우리가 다르게 하려는 점
- 모든 식당 메뉴를 한 화면에서 비교
- CCTV/센서 기반 실시간 혼잡도 제공
- 개인 취향 기반 AI 추천

## 기술/구현 방향
- 모바일 앱 (Flutter)
- 학교 식당 API 연동 또는 크롤링
- 혼잡도: 학교 측 협조 받아 실시간 데이터 수집
