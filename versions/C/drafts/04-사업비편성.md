# 4. 사업비 편성

> 담당: 재무 (또는 PM 겸임) | 금액 기준: 최대 300만원
> 분량: 0.5페이지 (표 위주)

## 사업비 내역

| 항목 | 세부 내용 | 금액 | 비고 |
|------|----------|------|------|
| 회식비 | 팀 회의 식비 | 30만원 | 1인 1회 5만원, 한도 30만원 |
| 사무비 | 사무용품 구입 | 30만원 | 한도 30만원 |
| 홍보비 | 포스터, 현수막 제작 | 50만원 | 소모성 물품 |
| 마케팅 대행 | (구체적 산출물 기재) | 110만원 | 전자세금계산서 필수 |
| 디자인 대행 | (구체적 산출물 기재) | 80만원 | 전자세금계산서 필수 |
| **합계** | | **300만원** | |

> financial-analyst 에이전트 활용: 예산 검증을 요청하면 규정 부합 여부를 자동 체크합니다.

## 주의사항
- 재료비: 시제품 개발용 소모품만 가능. 노트북 등 기자재 구입 불가
- 외주용역비: 팀이 직접 못 하는 것만 가능. 반드시 전자세금계산서 발행
- 외주 항목은 구체적으로 기재 (예: "SNS 콘텐츠 제작 외주 20건")
