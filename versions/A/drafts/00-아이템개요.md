# 사업 아이템 개요

| 항목 | 내용 |
|------|------|
| **팀명** | JClaw |
| **아이템명** | JClaw - 웹 기반 AI 에이전트 빌더 for 전북대 |
| **아이템 개요** | 교내 OpenStack + LLM API 위에 웹 No-Code 빌더를 올려, 전북대 2만 구성원 누구나 AI 에이전트를 만들고 Mattermost에서 즉시 사용하게 하는 플랫폼 |

## 주요 기능

| 기능 | 설명 |
|------|------|
| **No-Code 에이전트 빌더** | 웹 브라우저에서 드래그앤드롭과 설정 입력만으로 AI 에이전트를 생성. 프롬프트 템플릿, 도구 연결, 트리거 설정을 시각적 인터페이스로 제공하여 비전공자도 5분 내 첫 에이전트 완성 |
| **Mattermost 원클릭 배포** | 생성 완료 버튼을 누르면 에이전트가 전북대 Mattermost에 봇으로 자동 등록. 별도 서버 설정이나 API 키 발급 없이 채팅방에서 즉시 호출 가능 |
| **스마트 LLM 할당 시스템** | 교내 LLM API의 제한된 자원을 사용자별로 공정하게 할당. 실시간 사용량 대시보드, 할당량 초과 알림, 경량 모델 자동 전환으로 전체 서비스 안정성 확보 |

## 활동 목표

전북대에는 OpenStack 클라우드와 LLM API라는 강력한 AI 인프라가 이미 구축되어 있다. 그러나 이 인프라를 실제로 활용하는 학생은 극소수다. CLI를 다룰 줄 아는 전공자만이 접근 가능하기 때문이다.

JClaw는 이 **"인프라는 있지만 사용자가 없는"** 문제를 해결한다. 컴퓨터공학부 4명이 웹 기반 빌더를 개발하고, Mattermost 연동을 구현하여, 기술 장벽을 제거한다. **목표: 8개월 내 활성 사용자 1,000명, 에이전트 500개 생성, 비전공자 비율 50% 이상 달성.**
