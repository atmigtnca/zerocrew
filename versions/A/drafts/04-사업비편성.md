### 1. 재료비 (900,000원)

| 품목 | 산출근거 | 금액 |
|------|---------|------|
| 외부 클라우드 테스트 환경 (AWS/GCP) | 월 80,000원 × 5개월 — 교내망 외부에서 접근성 테스트, 부하 테스트용 | 400,000원 |
| 외부 LLM API 크레딧 | 월 60,000원 × 5개월 — OpenAI/Anthropic API로 교내 LLM 대비 품질 비교 테스트 | 300,000원 |
| 도메인/SSL 인증서 | 1년분 — 서비스 도메인 및 HTTPS 인증 | 50,000원 |
| 개발용 소모품 (케이블, 어댑터 등) | 시연 및 개발 환경 구성용 소모성 물품 | 150,000원 |
| **소계** | | **900,000원** |

## 예산 운영 원칙

1. **인프라 비용 0원**: 교내 OpenStack + LLM API 무상 활용으로 서버/모델 비용 절감 (8개월 기준 약 240~400만원 절감 효과)
2. **외주 집중 투자**: 전체 예산의 50%를 UI 디자인에 투입 — 비전공자 대상 서비스의 성패는 UI가 결정
3. **규정 준수**: 기자재(PC, 모니터 등) 및 인건비 편성 불가 규정 엄수
4. **증빙 관리**: 법인카드 또는 계좌이체 우선, 외주용역은 전자세금계산서 + 용역 계약서 + 결과물 확보
5. **집행 시기**: 사업 기간(5~12월) 내에만 집행, 정산 마감일 준수
