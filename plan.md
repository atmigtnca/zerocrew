아이템명: JClaw

한 줄 소개: 전북대 학생 누구나 웹에서 AI 에이전트를 만들고 Mattermost로 바로 사용할 수 있는 플랫폼

해결하는 문제: OpenClaw 같은 오픈소스 AI 에이전트 도구는 CLI 기반이라 비전공자가 사용하기 어려움. 학교에서 이미 OpenStack 클라우드와 LLM API를 학생들에게 충분히 제공하고 있는데 활용이 안 되고 있음

핵심 기능:
- 웹 UI에서 클릭 몇 번으로 AI 에이전트 생성
- 생성된 에이전트가 Mattermost에 자동 연결
- 교내 LLM API 사용량 할당/관리 (제한적이지만 충분한 양)

## 상세 정보 (선택 — 채울수록 결과가 좋아집니다)

### 왜 이걸 만들려고 하는지
교내 OpenStack 클라우드와 LLM API가 이미 학생들에게 제공되고 있지만, 기존 AI 에이전트 도구(OpenClaw 등)는 CLI 기반이라 비전공자가 접근하기 어렵다. 전북대 학생 모두가 AI 에이전트를 손쉽게 사용할 수 있도록 웹 기반 플랫폼을 만들고자 한다. 최소 1,000명 이상 사용자 확보가 목표다.


### 현재 사람들이 이 문제를 어떻게 해결하는지
- CLI 기반 오픈소스 도구(OpenClaw 등)를 직접 설치하고 설정 — 전공자만 가능
- ChatGPT, Claude 등 외부 상용 AI 서비스 개인 유료 구독 — 비용 부담
- AI 에이전트 기능 자체를 포기하고 단순 챗봇 수준에서만 사용


### 우리가 다르게 하려는 점
- 교내 인프라(OpenStack, LLM API) 활용으로 학생에게 비용 부담 없음
- 웹 기반 UI로 클릭 몇 번만으로 에이전트 생성 — 비전공자도 가능
- Mattermost 연동으로 학생들이 이미 익숙한 환경에서 즉시 사용 가능
- 외부 서비스 의존 없이 교내 폐쇄망에서 운영 가능하여 보안성 확보


### 기술/구현 방향
- 프론트엔드: 웹 기반 (React 등) — 클릭 기반 에이전트 빌더 UI
- 백엔드: 교내 OpenStack 클라우드 위에 배포
- LLM 연동: 교내 LLM API (사용량 할당/관리 기능 포함)
- 메신저 연동: Mattermost Bot API를 통해 생성된 에이전트 자동 등록
