# 예산 편성 가이드 (예비창업동아리 지원사업)

## 예산 총액
**300만원** (초과 불가)

## 현재 편성안
| 항목 | 금액 | 분류 |
|------|------|------|
| 회식비 | 30만원 | 회의비 |
| 사무비 | 30만원 | 사무용품비 |
| 홍보비 (포스터, 현수막) | 50만원 | 재료비/인쇄비 |
| 마케팅 대행 | 110만원 | 외주용역비 |
| 디자인 대행 | 80만원 | 외주용역비 |
| **합계** | **300만원** | |

## 항목별 집행 규정

### 회의비 (회식비)
- **한도**: 총 30만원
- **1인 1회**: 5만원 이내
- **증빙**: 카드전표 + 참석자 명단
- **주의**: 주류만의 회식 불가, 식사 포함 필수
- **팁**: "팀 회의 식비"로 사유 작성

### 사무용품비
- **한도**: 30만원
- **가능**: 문구류, 프린트 용지, 토너, USB 등 소모품
- **불가**: 노트북, 모니터, 키보드 등 기자재
- **증빙**: 카드전표/영수증

### 재료비 (홍보비 포함)
- **가능**: 시제품 재료, 포스터 인쇄, 현수막, 명함, 스티커 등 소모성 물품
- **불가**: 기자재 (노트북, 태블릿, 카메라 등)
- **증빙**: 카드전표/영수증
- **팁**: 항목을 구체적으로 (예: "A2 포스터 50매 인쇄")

### 외주용역비
- **핵심**: 전자세금계산서 필수 발행
- **가능**: 팀이 직접 수행 불가능한 전문 업무
- **불가**: 팀원이 할 수 있는 업무를 외주 주는 것
- **구체화 필수**:
  | ❌ 모호함 | ✅ 구체적 |
  |----------|----------|
  | 마케팅 대행 | SNS 콘텐츠 제작 외주 (인스타 피드 20건 + 릴스 5건) |
  | 디자인 대행 | 브랜드 디자인 외주 (로고 1종 + 명함 + 앱 UI 5화면) |
  | 영상 제작 | 서비스 소개 영상 촬영/편집 외주 (1분 영상 2건) |
- **증빙**: 전자세금계산서 + 용역 계약서 + 결과물

## 금지 항목 ❌
| 항목 | 이유 |
|------|------|
| 기자재 구입 (노트북, 태블릿 등) | 원천 불가 |
| 인건비 | 대부분 예비창업동아리에서 불가 |
| 경조사비 | 사업 무관 |
| 접대비 | 사업 무관 |
| 해외 출장비 | 예비창업 단계 부적절 |

## 자금조달 계획 프레임워크

### 3시점 구조
| 시점 | 설명 | 예시 |
|------|------|------|
| **과거** | 이전 투자/지원금 | 학생팀은 0원이 정상 |
| **현재** | 본 지원사업 + 자부담 | 300만원 + 팀원 출자 |
| **미래 1년** | 후속 자금 조달 계획 | 창업경진대회, 추가 지원사업 |

### 후속 자금 소스 (학생팀 활용 가능)
| 소스 | 금액 범위 | 시기 | 비고 |
|------|----------|------|------|
| 교내 창업경진대회 | 100~500만원 | 수시 | 대학마다 다름 |
| 창업진흥원 초기창업패키지 | 최대 1억원 | 매년 상반기 | 예비/초기 창업자 |
| 지역 창업지원센터 | 100~1,000만원 | 수시 | 전북창조경제혁신센터 등 |
| TIPS (민간투자주도형) | 최대 5억원 | 수시 | 기술 기반 스타트업 |
| 교내 창업보육센터 | 사무 공간 + 멘토링 | 수시 | 간접 지원 |

## 정산 주의사항
- 📅 사업 기간 내 집행 건만 인정
- 💳 법인카드 또는 계좌이체 우선 (현금 지양)
- 🧾 건별 증빙 보관 (카드전표, 세금계산서, 견적서)
- 📋 지출결의서 작성 (사업비 관리 양식)
- ⏰ 정산 마감일 준수 (보통 사업 종료 후 1개월)
