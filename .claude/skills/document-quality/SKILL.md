# 사업계획서 품질 기준

## 포맷 규격
| 항목 | 기준 |
|------|------|
| 서체 | 중고딕 12pt |
| 줄간격 | 160% |
| 분량 | 요약서 1p + 본문 5p 이내 (총 6p) |
| 여백 | 기본 (상하좌우 20mm) |

## 분량 배분 가이드
| 섹션 | 페이지 | 글자 수 (추정) | 비고 |
|------|--------|---------------|------|
| 0. 아이템개요 | 1p (별도) | 400~500자 + 표 | 심사위원 첫인상 |
| 1. 팀구성 | 0.5p | 200~300자 + 표 | 간결하게 |
| 2. 문제인식 | 2.5p | 1,200~1,500자 + 표/도표 | 핵심 섹션, 가장 공들일 것 |
| 3. 실현가능성 | 1.5p | 700~900자 + 표 | 로드맵+비교표 중심 |
| 4. 사업비편성 | 0.5p | 200~300자 + 표 | 표 위주 |

> 중고딕 12pt, 줄간격 160% 기준. 표가 많으면 글자 수 줄어듦.

## 필수항목 체크리스트

### 0. 아이템개요 (요약서)
- [ ] 팀명
- [ ] 아이템명
- [ ] 아이템 개요 (한 줄 소개)
- [ ] 주요 기능 (2~3개)
- [ ] 활동 목표

### 1. 팀구성
- [ ] 창업자(팀장) 역량 서술
- [ ] 팀원 테이블 (이름, 학과/학년, 역할, 전문성)
- [ ] 팀원 3인 이상
- [ ] 지도교수 포함
- [ ] 팀 시너지 효과 서술

### 2. 문제인식
- [ ] 배경 및 동기 (구체적 사례)
- [ ] 목적 및 필요성
- [ ] 시장 규모 (TAM/SAM/SOM 수치)
- [ ] 시장 성장률/트렌드
- [ ] 데이터 출처 명시
- [ ] 시장 진입 전략

### 3. 실현가능성
- [ ] 사업화 전략 (4단계 로드맵, 4~11월)
- [ ] 경쟁사 비교 테이블 (2개 이상)
- [ ] 차별화 포인트
- [ ] 자금조달 계획 (과거/현재/미래)

### 4. 사업비편성
- [ ] 항목별 금액 테이블
- [ ] 합계 = 300만원
- [ ] 항목별 세부 내용
- [ ] 외주 항목 구체적 산출물 명시

### 공통
- [ ] "작성방법" 안내 표 삭제
- [ ] 아이템명 전 섹션 통일
- [ ] 팀명 전 섹션 통일
- [ ] 숫자/통계 출처 표기

## 심사 기준 (추정 배점)
| 항목 | 배점 | 핵심 평가 포인트 |
|------|------|----------------|
| 문제인식 | 25% | 시장 이해, 데이터 근거, 진입 전략 |
| 실현가능성 | 25% | 로드맵 현실성, 경쟁력, 자금 계획 |
| 팀구성 | 20% | 역량, 역할 분담, 시너지 |
| 아이템개요 | 15% | 명확성, 임팩트, 차별성 |
| 사업비 | 15% | 적정성, 규정 준수, 구체성 |

> 문제인식 + 실현가능성 = 50%. 이 두 섹션에 가장 공을 들여야 합니다.
