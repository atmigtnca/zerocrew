# Competitive Analysis Framework

## Competition Types

### Direct Competition
Services solving the same problem in the same way.
- Example: 배달의민족 vs 쿠팡이츠 (both food delivery apps)

### Indirect Competition
Services solving the same problem differently, or different problems the same way.
- Example: 배달의민족 vs convenience store meals (same need, different approach)

### Alternatives
How users currently solve the problem without our service.
- Example: Excel, KakaoTalk group chats, manual processes, paper

> "No competitors" is never a valid conclusion. At minimum, analyze alternatives.

## Positioning Map Design

### Axis Selection Guide (2 meaningful axes)
Good axes should:
1. Be criteria users actually compare
2. Show clear differences between competitors
3. Allow us to claim an advantageous position

### Axis Examples
| X-axis candidates | Y-axis candidates |
|-------------------|-------------------|
| Price (low↔high) | Features (simple↔complex) |
| Ease of use (complex↔simple) | Specialization (general↔expert) |
| Target (individual↔enterprise) | Customization (standard↔custom) |

## Differentiation Point Extraction

### Differentiation Types
1. **Feature differentiation**: Core feature competitors lack
2. **UX differentiation**: Same features but better experience
3. **Target differentiation**: Focus on segment competitors ignore
4. **Price differentiation**: Free/low-cost or premium strategy

### Differentiation Sentence Formula
❌ "We fill a gap in the market where no competitor exists" (claims no competition)
✅ "While competitor A focuses on [feature X], we provide [value W] to [target Z] through [feature Y]"
