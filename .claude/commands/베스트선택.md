내 3개 버전을 비교해서 가장 좋은 1개를 선택해줘.

팀원 알파벳: $ARGUMENTS

비교 대상:
- versions/$ARGUMENTS-1/drafts/
- versions/$ARGUMENTS-2/drafts/
- versions/$ARGUMENTS-3/drafts/

각 버전을 전체 통독하고, 아래 기준으로 비교해줘:
1. 스토리 일관성: "문제 → 해결 → 시장 → 전략 → 예산"이 하나로 이어지는가?
2. 데이터 근거: 시장 수치에 출처가 있는가?
3. 차별화 설득력: 왜 우리 팀이어야 하는지 납득되는가?

가장 좋은 버전을 versions/$ARGUMENTS/drafts/ 에 복사해줘.
왜 이 버전을 선택했는지 간단히 설명해줘.
